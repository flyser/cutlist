
A simple script to losslessly cut a large video file in many smaller ones.

### Usage ###

Just call `cutlist.sh` using two parameters. The first one being the video file
you want to cut and the second being a file containing timecodes as described
below:

### Input file syntax ###

See `example-cutlist.txt`.


### Configuration ###

You can configure some things in the ~/.config/cutlist/cutlistrc file. See
the beginning of `cutlist.sh` for more information.
